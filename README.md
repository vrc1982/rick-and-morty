
# Rick And Morty Challenge

Creado por Victor Rojas


## Ejecutar localmente

Clonar el proyecto

```bash
  git clone https://github.com/vrc1982/rick-n-morty.git
```

Ir al directorio del proyecto

```bash
  cd rick-n-morty
```

Instalar dependencias

```bash
  npm install
```

Iniciar el servidor

```bash
  npm start
```
